# 会出海客户审核交付包

日期：2026-07-05  
审核站地址：https://hch.ideaegg.com.cn

## 1. 当前状态说明

当前站点是客户审核站，不是正式生产发布。

- 用途：用于客户、Chris 和运营团队审核品牌表达、客户路径、AI 顾问体验和业务口径。
- 域名性质：`hch.ideaegg.com.cn` 是临时审核域名，不作为长期生产域名资产。
- 搜索收录：当前为 `noindex / nofollow`，`robots.txt` 禁止抓取，`sitemap.xml` 为空。
- 数据状态：当前仍包含 MOCK 数据和示例内容，不代表真实资源、真实报价、真实档期或正式服务承诺。
- 能力状态：AI 顾问为 Phase 1 受控 Agent / 规则与 MOCK 能力，不是完整真实 AI provider + 真实业务系统闭环。

客户审核时请重点看“是否表达正确、路径是否清楚、客户是否愿意继续咨询”，不要把当前站点当作最终上线版本验收。

## 3. 页面审核重点

### 首页

- “会出海，帮中国企业在东南亚，把一场会办得更省心”是否准确。
- 视觉是否体现商务可信、深海蓝 / teal / 暖金方向，而不是旅游网站或泛 SaaS 后台。
- 五大服务场景、精选场地、会务物料与视觉支持、服务流程、FAQ / GEO 入口是否完整。
- CTA “免费提交办会需求”和“问问 AI 办会顾问”是否足够清楚。

### AI 顾问

- 是否像“会务顾问”而不是普通聊天框或报价表。
- 模糊咨询时是否先回答问题，再逐步收窄范围。
- 是否明确会务地点 / 目标城市，并能承接活动意图、规模、预算和服务偏好。
- 是否避免直接承诺正式价格、实时档期、付款和取消条款。
- AI 语气是否稳重、可信、简洁，不像个人 IP、不像客服脚本。

### 方案配置

- 包型名称是否符合已定口径：经济型 / 标准型 / 高配型。
- 服务项是否覆盖：场地、晚宴、茶歇、AV/舞台、会议物料、接送机、住宿、同传、摄影摄像。
- 预算展示是否明确是预算结构估算 / 参考范围，不是正式报价。
- 调整服务项和切换包型后，预算、服务建议和需确认事项是否能被理解。

### 询盘与成功页

- 表单是否足够轻量，不让客户一开始承担过多填写成本。
- 联系方式、活动时间、人数、活动类型是否是必需信息。
- 成功页是否清楚说明后续会由顾问确认正式报价、档期、付款和合同条款。

### Ops 预览页

- 资源主档是否清楚区分参考价范围与动态影响因素。
- 当次询价单是否清楚表达“本次活动下的实时档期和价格确认”。
- 内部线索页是否只作为运营页，不从客户入口暴露。

## 5. 客户侧严禁暴露内容

客户侧页面、客户可见 API、客户截图材料中不应出现：

- 真实性判断、真实性评分、`authenticity_score`
- 意向强度、意向评分、`intent_score`
- 线索优先级、`lead_priority`
- 风险标记、`risk_flags`
- 内部跟进建议、推荐开场白、推荐回复
- 供应商内部谈判备注、底价、返点、议价空间
- 供应商原始响应摘要、档期冲突内部说明
- Chris / 运营内部判断客户值不值得跟进的字段

客户侧可以看到的是：已确认需求摘要、咨询进度、方案包选择、服务项选择、预算参考范围、预算覆盖说明、假设前提、未包含项、需要人工确认的事项和提交状态。

## 6. 需要 Chris / 客户补充的真实资料

为了从审核站进入真实业务闭环，需要补充：

- 城市资源：吉隆坡、槟城、新山、新加坡等目标城市的真实资源清单。
- 合作方资料：场地、酒店、晚宴、AV、物料、接送、翻译、摄影摄像供应商。
- 参考报价：按资源类型、规模、淡旺季、提前期和档期敏感度整理参考区间。
- 适用条件：容量、最低消费、提前预订期、服务范围、适合活动类型。
- 常见问题：客户常问的预算、签证、接送、搭建、合同、付款和取消条款。
- 典型客户案例：可公开使用的行业、活动规模、城市、成果和图片素材。
- 联系方式：正式对外联系人、邮箱、电话、WhatsApp / 微信机制。
- 运营机制：谁接收询盘、谁确认报价、多久响应、如何升级给 Chris。
- 品牌素材：正式矢量 logo、品牌规范、真实照片和可公开案例素材。

## 7. 建议审核反馈格式

请按下面格式反馈，便于拆分优先级和实施：

| 备注 | 相关截图、参考链接、联系人或决策人 |

## 8. 客户审核结论建议

本轮建议客户重点确认：

1. 品牌方向和首页主张是否成立。
2. AI 顾问是否符合“先咨询、再整理方案、最后人工确认”的业务方式。
3. 包型、服务项和预算解释是否符合 Chris 的真实销售口径。
4. 询盘表单和下一步说明是否足够清楚。
5. 真实资料补充清单是否完整。

不建议在本轮确认：

- 正式 SEO 收录。
- 正式生产域名。
- 真实报价承诺。
- 登录权限、真实通知、真实数据库和完整 AI provider 生产能力。
